# AI Service Frontend (Vite + React)

This is a minimal scaffold for the category/subcategory UI discussed.

## Setup
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Build: `npm run build`
4. Preview production build: `npm run preview`

Files included:
- src/main.jsx
- src/App.jsx
- src/data/categories.js
- src/components/{CategoryGrid,CategoryCard,CategoryPanel}.jsx
- src/styles/styles.css
